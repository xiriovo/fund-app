# 功能介绍

## 核心功能

### 自选管理

- 添加/删除自选基金
- 实时估值自动刷新
- 支持搜索基金代码或名称
- 左滑快速删除

### 实时估值

- 交易时间内秒级刷新
- 显示估算净值和涨跌幅
- 价格变动闪烁提示（模仿交易所效果）

### 曲线图表

- 分时走势图
- 日/周/月/年 曲线
- 专业的图表交互体验

### 持仓管理

- 记录买入成本和份额
- 自动计算持有收益
- 支持 A类/C类基金费用计算
- 当日盈亏实时更新

### 智能提醒

- 涨跌幅提醒
- 净值提醒
- 支持浏览器通知

## 进阶功能

### 基金对比

多基金走势对比分析，计算以下指标：
- 累计收益率
- 最大回撤
- 波动率
- 夏普比率

### 投资回测

- 定投回测模拟
- 一次性买入回测
- 自定义投资周期

### 收益报告

生成可分享的收益报告图片，包含：
- 总资产和收益
- 持仓分布
- 收益趋势图

### 投资日历

- 记录重要投资事件
- 分红/开放日提醒
- 自定义备忘录

## 市场行情

### 大盘指数

- 上证指数
- 深证成指
- 创业板指
- 沪深300

### 基金排行

- 日/周/月/年涨幅排行
- 基金经理排行

### 基金筛选

- 按类型筛选（股票型/混合型/债券型等）
- 按涨跌幅筛选
- 按规模筛选

## 个性化

### 主题切换

- 浅色模式
- 深色模式
- 跟随系统

### 数据本地存储

- 自选列表本地保存
- 持仓数据本地保存
- 无需登录，保护隐私
